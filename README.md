# Budget Planner

A simple Budget Planner web application built using React.  
This project helps users track their income and expenses and manage their monthly budget efficiently.

---

## Features

- Add Income
- Add Expenses
- View Budget Summary
- Track Expense List
- Shows remaining balance
- Warning when budget exceeds

---

## Tech Stack

- React
- Redux
- JavaScript
- CSS
- Vite

## How to Run the Project

1. Clone the repository

git clone https://github.com/DevendraSingh5377/miniProject_budgetplanner

2. Install dependencies

npm install

3. Start the development server

npm run dev
